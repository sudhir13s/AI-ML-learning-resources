# Data Visualization — Matplotlib · Seaborn · Plotly
> The three pillars of Python plotting: **Matplotlib** (the low-level engine everything builds on),
> **Seaborn** (statistical charts in one line, on top of Matplotlib), and **Plotly** (interactive,
> web-native figures). Together they cover everything from a quick EDA histogram to a polished
> interactive dashboard.

**Why it matters:** exploratory data analysis is the first thing you do on any dataset, and
communicating results is the last. Knowing when to reach for Matplotlib's control vs Seaborn's
brevity vs Plotly's interactivity — and the figure/axes mental model — is a core practical skill.

**⭐ Start here — suggested path:**

1. **Learn the figure/axes model** — read the Matplotlib [Quick start guide](https://matplotlib.org/stable/users/explain/quick_start.html). *Understanding Figure vs Axes is the key that unlocks all of Matplotlib.*
2. **Watch it come together** — [Matplotlib Crash Course](https://www.youtube.com/watch?v=3Xc3CA655Y4) (freeCodeCamp). *A fast, complete tour of the most common plot types.*
3. **Get statistical plots for free** — work the [Seaborn tutorial](https://seaborn.pydata.org/tutorial.html). *Seaborn turns DataFrames into rich statistical charts with almost no code.*
4. **Add interactivity** — skim [Plotly fundamentals](https://plotly.com/python/plotly-fundamentals/), then [Intro to Plotly Express](https://www.youtube.com/watch?v=b1Fyf83eWnM). *Plotly Express gives interactive, shareable figures from one function call.*
5. **Keep galleries handy** — bookmark the [Matplotlib gallery](https://matplotlib.org/stable/gallery/index.html) and copy-adapt. *Finding the closest example and tweaking it is the fastest real workflow.*
